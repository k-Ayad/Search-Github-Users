Welcome to

our Angular GitHub User Search project, a dynamic web application that combines the power of Angular, RxJS, and NgRx to deliver a seamless and efficient experience for searching GitHub users.

Lets Start

Key Features

GitHub User Search Our application allows you to search for GitHub users effortlessly. Just type a username in the search bar, and our application fetches the relevant user information in real-time.
Asynchronous Operations We leverage RxJS, a powerful library for reactive programming, to handle asynchronous operations. This ensures a smooth and responsive user experience while fetching data from GitHub's servers.
State Management with NgRx Managing application state is a breeze with NgRx. We use NgRx to store and manage the application's data, making it easy to keep track of user search results and maintain a consistent user interface.
